A simple 2D platform game written in C++ language using NCurses library

## Requirements
```
NCurses librarys for C++
g++ compiler
MakeFile extencion

## Usage
```
(linux)
$ make all
$ ./main

(windows)
$ make all
$ main.exe

## Controls
```
* Move - arrow keys
* Shoot - space
* Jump - KEY_UP
* Pause - Esc
* Confirm (in menu) - 'k'
* Move (in menu) - KEY_UP/KEY_DOWN

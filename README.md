A simple 2D platform game written in C++ code using NCurses library

## Requirements
NCurses librarys for C++

## Usage
```
(linux)
$ make all
$ ./main
```
(windows)
$ make all
$ main.exe

## Controls
* Move - arrow keys
* Shoot - space
* Pause - Esc
* Confirm (in menu) - 'k'
* Move (in menu) - KEY_UP/KEY_DOWN
